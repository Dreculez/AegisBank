## Approche "Mobile First"
Conformément au cahier des charges, le développement a privilégié :
* **Performance :** Code léger pour un chargement rapide (MVP).
* **Adaptabilité :** Design pensé pour les petits écrans avant d'être étendu au desktop.
* **Accessibilité :** Structure HTML sémantique pour un meilleur référencement et usage.
